Fandango Clone

Led a team of four in the design and development of a full-stack Java Swing movie ticketing application that replicated core functionality of commercial booking platforms. Architected and implemented the graphical user interface, business logic, and a custom object-serialization persistence layer using vanilla Java without external frameworks. Designed a reservation engine supporting real-time seat locking, multi-threaded client connections, conflict prevention, and a dynamic demand-based ticket pricing algorithm to ensure data consistency and scalability in a concurrent environment.

Java NLP Similarity Engine

Developed a Java-based natural language processing application for document similarity analysis using classical information retrieval techniques. Implemented k-shingle generation, TF–IDF vectorization, and multiple similarity metrics to compare long-form documents with high accuracy. Extended the system beyond document-level comparison by incorporating phrase-level similarity detection, generating an interactive HTML heatmap that localized overlapping content and provided an interpretable visualization of semantically similar text regions.

NutriSync

Developed NutriSync, a cross-platform fitness tracking application that enables users to log workouts, monitor long-term exercise progress, maintain workout streaks, and visualize performance through an interactive dashboard. Designed the application using an object-oriented architecture with reusable page components, asynchronous database operations, and a relational database schema implemented with Prisma ORM and SQLite to ensure scalable, type-safe data management. Implemented secure user authentication with BCrypt password hashing, a rule-based achievement engine that evaluates workout milestones and streaks in real time, and dynamic progress visualizations that provide personalized performance insights through responsive UI components. The application also incorporates comprehensive input validation, timezone-aware streak calculations, and asynchronous data loading to deliver a responsive and reliable user experience.

Fuzzy Pseudocode Compiler

Developed a deterministic compiler that translates human-readable pseudocode into executable Python through a complete compiler pipeline. Designed modules for lexical analysis, fuzzy pattern matching, parsing, Abstract Syntax Tree (AST) construction, and Python code generation, enabling flexible natural-language-inspired syntax while producing valid Python programs. Implemented support for variables, arithmetic expressions, conditionals, loops, and nested control structures, along with a command-line interface for compiling and executing pseudocode locally. The project explores compiler construction, programming language design, and deterministic parsing without relying on machine learning or external services.

Over-the-board chess engine

Developed a client–server chess application that synchronizes game state between two connected clients for over-the-board multiplayer play. Built the underlying chess engine responsible for legal move generation, rule enforcement, turn management, and board synchronization, while also implementing a computer-controlled opponent for single-player gameplay.
